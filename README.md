# PublishingWithArcpy
For UC demonstration
1. Download the .7z onto local machine, and unzip it. cd to the publishingSamples2017 folder.
2. Modify the secure\AGO_pass.txt with your own username and password
3. Rename the service name in the PublishLayersAsWTL.py and PublishLayersAsWFL.py files
4. Sign into online with username and password, then run the two python files to publish SD files and create services.
